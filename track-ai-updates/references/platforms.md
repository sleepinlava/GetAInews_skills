# 平台来源地图

当用户只说“抓 AI 最新进展”而没有指定范围时，优先从下面这些地方找高信号内容。

## X

优先关注以下账号类型：
- OpenAI、Anthropic、Google DeepMind、Meta AI、Mistral、xAI、Hugging Face 等官方账号
- 创始人、研究负责人、模型作者、核心工程师
- 高频发布演示和开源项目的开发者
- 在某个主题下被多人连续转发和引用的关键账号

可用检索思路：
- `site:x.com AI model release`
- `site:x.com Hugging Face new model`
- `site:x.com AI agent demo`
- `site:x.com open source llm release`

## Hugging Face

优先查看：
- Models
- Spaces
- Papers
- Datasets
- Blog

适合关注的迹象：
- 新模型上线
- Space 演示快速传播
- 与热门论文或仓库互相链接
- 同一主题在短时间内出现多个相关项目

可用检索思路：
- `site:huggingface.co huggingface.co/models reasoning`
- `site:huggingface.co huggingface.co/spaces video generation`
- `site:huggingface.co huggingface.co/papers agent`

## 官方来源

常见高信号域名：
- `openai.com`
- `anthropic.com`
- `deepmind.google`
- `ai.meta.com`
- `mistral.ai`
- `x.ai`
- `huggingface.co/blog`

重点看：
- 发布博客
- release notes
- changelog
- docs 更新
- 模型说明与价格变化

## GitHub

适合核实：
- 开源仓库是否真实活跃
- 是否刚发了 Release
- 是否出现 star 快速上涨
- 是否存在可运行 demo 或完善文档

## 选题建议

当用户没有给主题时，优先从这些方向挑：
- Coding agents
- 多模态模型
- 视频生成
- 推理模型
- 开源 LLM
- 评测与 benchmark
- 推理加速与部署工具

## 单条快讯模板

每条建议包含：
1. 标题
2. 日期
3. 平台
4. 核心变化
5. 影响判断
6. 来源链接
7. 可信度备注（可选）
